Claude API (AI Layer)
역할: 고객 문의 의도 분류 & FAQ 매칭
Model: claude-sonnet-4-6 (비용 효율)
Input: 고객 문의 원문
Output: { category, confidence, suggested_template_id }
Threshold: confidence ≥ 0.85 → 자동답변
confidence < 0.85 → 사람 에스컬레이션
System Prompt: "옥수주조 CS 어시스턴트. 막걸리 관련 문의를
6개 카테고리(유통기한/보관/도수/배송/선물/기타)로 분류하세요."
인증 & 환경 설정

client_id / client_secret → n8n Credential에 저장

Rate Limit: ~2 req/sec (내스토어 앱 기준)

5
